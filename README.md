## Онлайн платформа-торговой сети электроники

### ## Ветка feature/admin-network-tools

В данной ветке реализованы дополнительные требования к административной панели Django для модели звена сети.

### Что добавлено

1. Кликабельная ссылка на поставщика

В списке объектов сети в админке вместо обычного текстового отображения поставщика реализована кликабельная ссылка, ведущая на страницу редактирования поставщика.

Это позволяет быстро переходить по цепочке поставщиков.

2. Фильтрация объектов

В админке реализованы фильтры:

- по городу
- по стране
- по типу звена сети

3. Admin action

Добавлено массовое действие:

**Очистить задолженность перед поставщиком**

Действие позволяет обнулить задолженность у выбранных объектов сети.

4. Поля только для чтения:

- `hierarchy_level` (уровень иерархии)
- `created_at` (дата создания)

Так как они рассчитываются автоматически системой.

### Проверка

Функциональность проверена через Django Admin:

- переход по ссылке на поставщика работает
- фильтрация по городу работает
- массовое обнуление задолженности работает корректно
