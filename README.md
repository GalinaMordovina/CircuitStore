## Онлайн платформа-торговой сети электроники

### Ветка feature/network-models

В данной ветке реализованы основные модели предметной области.

### Модель Product

Хранит информацию о продуктах электроники:

- название
- модель
- дата выхода на рынок

### Модель NetworkNode

Представляет звено сети продаж электроники.

Содержит:

- название
- тип звена (завод, розничная сеть, индивидуальный предприниматель)
- контактные данные:
  - email
  - страна
  - город
  - улица
  - номер дома
- поставщика (ForeignKey на саму модель)
- продукты (ManyToMany связь с Product)
- задолженность перед поставщиком
- уровень иерархии
- дату создания

### Реализованная логика

- звено сети не может ссылаться само на себя как на поставщика
- завод не может иметь поставщика
- реализована защита от циклических ссылок
- уровень иерархии вычисляется автоматически на основе цепочки поставщиков

### Django Admin

Модели зарегистрированы в административной панели:

- Product
- NetworkNode

Настроены:

- отображение основных полей
- поиск
- фильтрация по стране, городу и типу звена
