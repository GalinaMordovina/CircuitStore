### Ветка swagger-and-docs

## Документация API (Swagger / ReDoc)

В проекте реализована автоматическая генерация документации API с использованием библиотеки **drf-spectacular**.

Документация формируется на основе сериализаторов, view и дополнительных аннотаций схем.

### Доступ к документации

Swagger UI:

[http://127.0.0.1:8000/api/docs/swagger/](http://127.0.0.1:8000/api/docs/swagger/)

ReDoc:  
[http://127.0.0.1:8000/api/docs/redoc/](http://127.0.0.1:8000/api/docs/redoc/)

OpenAPI schema:  
[http://127.0.0.1:8000/api/schema/](http://127.0.0.1:8000/api/schema/)

### Возможности документации

Swagger позволяет:

- просматривать все доступные API endpoints
- видеть параметры фильтрации, поиска и сортировки
- отправлять тестовые запросы напрямую из браузера
- проверять структуру запросов и ответов

### Документированные endpoints

В документации отображаются:

- `/api/health/` - проверка работоспособности API
- `/api/network-nodes/` - управление звеньями сети
- `/api/token/` - получение JWT токена
- `/api/token/refresh/` - обновление JWT токена

### Используемые технологии

Для генерации документации используется:

`drf-spectacular`

Документация автоматически обновляется при изменении сериализаторов и view.
